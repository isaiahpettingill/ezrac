# Agon Hex Diagnostics

A compact Agon MOS diagnostics page demonstrating `agon.text` hexadecimal formatting and MOS sysvars. It displays keyboard and mouse state captured at startup.

## Build

```sh
cargo run -- build examples/agon-mos/hex-diagnostics/src/main.ezra
```

The executable is written to `target/agonlight-mos-ez80/hex-diagnostics.bin`.
