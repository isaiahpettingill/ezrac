# ez180N Mandelbrot

A `ez180n-ez80` console demo that renders a block-glyph Mandelbrot-style image and presents it to the framebuffer.

Build from the repository root:

```sh
cargo run -- build examples/ez180n/mandelbrot/src/main.ezra
```

Run the generated `ez180n-mandelbrot.gaem` from `target/ez180n-ez80` with an ez180N-compatible core.
