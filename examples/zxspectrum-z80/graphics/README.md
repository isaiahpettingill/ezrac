# ZX Spectrum Graphics

Draws an alternating bitmap pattern across the Spectrum's non-linear display memory and assigns bright white ink on blue paper.

```sh
cargo run -- build examples/zxspectrum-z80/graphics/src/main.ezra
```

The resulting `zx-graphics.tap` is written under `examples/zxspectrum-z80/graphics/target/zxspectrum-z80`.
