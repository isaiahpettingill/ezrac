# TI-84 Plus CE Color Bars

Displays eight full-height RGB565 color bars and waits for a key before returning to TI-OS. This demonstrates the native 16-bit framebuffer, standard palette constants, rectangle filling, and OS input.

```sh
cargo run -- build examples/ti84plusce/color-bars/src/main.ezra
```

Run the resulting program with `Asm(prgmEZRABARS)`.
