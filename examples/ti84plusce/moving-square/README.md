# TI-84 Plus CE Moving Square

An interactive graphics example. Move the square with the arrow keys and press Clear to return to TI-OS. It demonstrates TI key constants, bounded movement, redraws, custom RGB colors, and clean shutdown.

```sh
cargo run -- build examples/ti84plusce/moving-square/src/main.ezra
```

Run the resulting program with `Asm(prgmEZRAMOVE)`.
